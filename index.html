How to use (setup)

  1. Drag bookmark to bookmark bar
  2. Navigate to target site
  3. Click on bookmark
  4. Set dataset name and click [Go]

How to use (wrapper generation)

  1. Select area that contains all data that needs to be extracted
  2. Right click and select object type (annotation)
  3. Select item inside
  4. Right click and select object type (annotation)
  5. Repeat ...
